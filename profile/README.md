# antora-supplemental

A GitHub organization for **supplemental features and tooling** around [Antora](https://antora.org). The Antora project does not maintain GitHub-specific assets such as Actions and repository templates; this org hosts those and related ecosystem projects.

## What we host

- **GitHub-specific assets** — Actions and workflow templates that Antora has declined to maintain
- **Fork-based development** — Forks of Antora dependencies for prototyping and submitting upstream changes
- **Ecosystem projects** — Supplemental tools and registries for the Antora community

## Notable repositories

| Repo | Description |
|------|-------------|
| [docs](https://github.com/antora-supplemental/docs) | Organization documentation and publishing guides |
| [antora-demo](https://github.com/antora-supplemental/antora-demo) | Demo Antora site and learning resource |
| [antora-github-action](https://github.com/antora-supplemental/antora-github-action) | Reusable GitHub Action to build Antora sites |
| [antora-build-action](https://github.com/antora-supplemental/antora-build-action) | Lean build action for Antora + GitHub Pages |
| [antora-github-deploy-template](https://github.com/antora-supplemental/antora-github-deploy-template) | Repository template for Antora → GitHub Pages |
| [antora-extensions-registry](https://github.com/antora-supplemental/antora-extensions-registry) | Antora Extensions Registry |
| [antora-dark-theme](https://github.com/antora-supplemental/antora-dark-theme) | Dark theme for Antora documentation sites |

Organization workflow templates live in the `workflow-templates/` directory of this repository (`.github`) and in [.github-private](https://github.com/antora-supplemental/.github-private) for private repos.
